Flash Card App Design


Previous Next

Description

This is a design I made for a flash card app. When the user opens the app, they would be greeted by the main menu screen (first slide). On this screen, the user can choose to either view the flash cards, create new ones, edit existing ones, or delete them. If the user chooses to create a new flash card, they would be taken to a screen where they can input the name of the card, the prompt, the and answer (second slide). When they are finished, they could either create the card or cancel the process if they're not satisfied with it and return to the main menu. If the user wants to edit a card, they would first need to choose a card (third screen). When the user selects the card they want to edit, they will be taken to a screen similar to the one where they could create one and edit the text (fourth slide). If the user wants to delete a card for any reason, they would be taken to a screen where would need to check the box next to the card they want to remove (fifth slide). Any cards with the corresponding boxes checked would be deleted when the user clicks the delete button.
