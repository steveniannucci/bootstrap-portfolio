Description

This is the design I came up with for the exercise app before I started developing it. The main menu (first slide) is the only screen that I didn't deviate from when making the final product. I originally assumed that there would be a duration, repitition, and resistance screen for each exercise. However, I was supposed to only have either a duration or repitition screen for each. I also planned for the user to be able to navigate between screens by tapping on the indicators at the bottom of the screen or swiping left or right. This wasn't necessary for the final product so I scrapped the idea. The Duration screen (second slide) was originally going to have two different options for resetting the timer. One option was to reset the timer to a certain time that was previously set by the user. The other option is to set the timer at 00:00:00. Only the latter option was needed.
