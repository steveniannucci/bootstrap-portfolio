Menu App



Description

This is a data-driven menu page I made using the React framework. It displays menu items from a Dinner and Dessert menu, respectively. Each menu item has a name, price, and description as well as a button that, when clicked, should add the corresponding item to a cart for checkout. A problem that I ran into while making the app was getting the sample data to display from the json. I solved by using the map function to iterate over the data and display it. I chose a simple design for the app with little to no styling to focus on the structure of the displayed data. The layout is admittedly not that great and it can use some adjustments.
