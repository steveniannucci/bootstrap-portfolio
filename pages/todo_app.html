Todo App


Description

This is a todo app I made that provides an interface for users to add and complete tasks. I built the app using the React Native framework and Expo CLI. The login and registration pages (not shown) are handled by Asynchronous Local Storage. When users are on the task list screen, they can enter input into the text fields for the task name and deadline. Once they click the Add Task button, the task will be added to the list as a container with a checkbox inside it. Checking the box will strike through the text within the container, marking the task as complete.
