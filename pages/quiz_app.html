Quiz App


Description

This is a simple quiz app I made using the React Native framework and Expo CLI that accepts a list of questions and answers and renders them to the screen. The user can click on one or more answers depending on the type of question for each question. Their score is calculated and the quiz results are displayed on a summary screen after all of the questions have been answered. If the user got the correct answer for a question, the answer is highlighted in green. Otherwise, their choice is highlighted in grey. Regardless, the choices they make are bolded to indicate that they're theirs. The correct answers to each question are checked. To clarify why an answer is correct, I added a brief explanation for each question. I thought adding that would make the quiz more engaging for the user.
